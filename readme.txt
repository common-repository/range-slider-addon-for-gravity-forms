== Description ==
A simple and nice plugin to add range slider to pick number or customize data easily. This plugin adds a responsive range slider field with smooth animation. The field comes with a lot of options that's controllable from the Gravity Forms UI.

== Features ==
* Add minimum and maximum number
* Show/hide picked data in tooltips.
* Add prefix in tooltips number
* Add step number.
* Added merge tag for calculation with conditional logic.
* Add default value (new).
* More feature coming...

= Requirements =

* PHP 5.6
* Gravity Forms 2.0.0, 2.5.0, 2.6.0 or higher

== Installation ==
Installation is fairly straight forward. Install it from the WordPress plugin repository.
